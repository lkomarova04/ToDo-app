# Todo App (React + TypeScript + Redux Toolkit + REST API)

## 📝 Описание 
Реализовано на **React** и **TypeScript**, с использованием **Redux Toolkit** для управления состоянием и **REST API** для сохранения задач на сервере.  
Приложение поддерживает:

- Добавление новых задач
- Удаление одной или всех задач
- Отметку задач как выполненных
- Фильтрацию по статусу: `all`, `active`, `completed`
- Темную и светлую тему с сохранением выбора в `localStorage`

---

## ⚡ Технологии

- React + TypeScript
- Redux Toolkit
- REST API (JSON Server)
